Part 1 - Basic Understanding of how the internet works

The Web refers to the 'World Wide Web', which is a collection of HTML (Hyper Text Markup Language) documents. Hyperlinks between pages create The Web. The major pieces of The Web are: people, computers, browsers, and the Internet. HTTP (the main protocol of the web) connects servers to the internet. Servers are optimised for hosting files. The internet uses HTTP to communicate with you, your browser, and servers. The HTML is the text content (what you see), markup (what content looks like), References to other documents (videos, images) and links to other pages.

Part 3 - CSS

CSS stands for 'Cascading Style Sheet'. It is used to apply formatting to your HTML page. The most specific rules are applied, and the code will keep going down and down into the rules, to the one that best describes the element, and that rule gets applied. You can write a selector, or select an HTML tag to apply styles to. For example:

.description {
    color:red;
}

You can change anything to do with the formatting, and standardise it through CSS. It is also possible to have multiple CSS files, so you can start with a library such as Boostrap, and then use your own for customisations. CSS is important because it allows you to not repeat yourself. If you would like to change something, it is very helpful if you only need to change it in one place!
